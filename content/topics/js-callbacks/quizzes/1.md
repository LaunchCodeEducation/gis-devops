## Theory

> A Higher Order Function satisfies which of the following conditions?

<!-- 
- receives another function as an argument
- returns a function as its output
-->

- receives another function as an argument
- contains recursive internal logic
- returns a function as its output
- receives anything except a function as an argument

> What feature of JavaScript enables the use of HOF?

<!-- 2. functions are first class citizens -->

1. JavaScript does not support HOF.
2. functions are first class citizens
3. anonymous functions
4. optional function arguments

## Usage

> What is responsible for invoking the `callback`?:

<!-- 2. the `procedure` function invokes it -->

1. You invoke it **before** invoking the `receiving` function.
2. The `receiving` function invokes it as part of its procedure.
3. You invoke it **after** invoking the `receiving` function.

> Information is **sent out** of a `receiving` function through:

<!-- 1. the arguments given to the `callback` when invoked -->

1. The arguments given to the `callback` when invoked.
2. The output returned from the `callback` when invoked.

> Information is **sent into** a `receiving` function through:

<!-- 1. the arguments given to the `callback` when invoked -->

1. The arguments given to the `callback` when invoked.
2. The output returned from the `callback` when invoked.

> Every `callback` must receive arguments from the `receiving` function.

<!-- 2. false -->

1. true
2. false
