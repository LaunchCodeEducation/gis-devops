### Array

> Arrays are JavaScript's list-like objects.

Arrays allow you to store any number of primitive literal values, variables, or objects as the **elements** of that array.

Arrays are indexed starting at 0.

___

#### Array Examples

```js
const someArray = ["a", 5, false];
```

someArray elements:

- 0: "a"
- 1: 5
- 2: false

```js
const someArray = ["a", 5, false];
console.log(someArray[0]); // prints "a"
console.log(someArray[1]); // prints 5
console.log(someArray[2]); // prints false
```

___

#### Array `length` Property

Arrays have a built in `length` property that will return the number of elements in the array.

```js
const someArray = ["a", 5, false];
console.log(someArray.length); // prints 3, the number of elements in someArray
```

___

#### Subarrays

An array can hold a different array as an element.

```js
const someArray = ["a", 5, false, [10, 9, 8]];
console.log(someArray[3]); // prints [10, 9, 8]
console.log(someArray[3][0]); // prints 10
console.log(someArray[3][1]); // prints 9
console.log(someArray[3][2]); // prints 8
```

### Non Iterative Array methods

- `.indexOf()`
- `.slice()`

___

#### `.indexOf()`

> Searches the array for a value, and returns the index of the first occurrence. If the value is not found it returns `-1`.

```js
const someArray = ["a", "b", "c"];
console.log(someArray.indexOf("a")); // prints 0
console.log(someArray.indexOf("c")); // prints 2
console.log(someArray.indexOf("dog")); // prints -1
```

___

#### `.slice()`

> Takes an array and returns a substring of the array starting at a specific index.

```js
const someArray = ["a", "b", "c"];
console.log(someArray.slice(0)); // prints ["a", "b", "c"]
console.log(someArray.slice(1)); // prints ["b", "c"]
console.log(someArray.slice(2)); // prints ["c"]
console.log(someArray.slice(1, 2)); // prints ["b"]
```

### Object Literal

> Object literal's are one of JavaScript's key value pair data structures.

___

#### Object Literal Example

```js
const someObject = {"name": "Bernie", "age": 5};
console.log(someObject.name); // prints "Bernie"
console.log(someObject.age); // prints 5
```

___

#### Object Static Methods

> The Object class has a few static methods that provide information about a provided object.

```js
const someObject = {"name": "Bernie", "age": 5};
console.log(Object.keys(someObject)); // prints ["name", "age"]
console.log(Object.values(someObject)); // prints ["Bernie", 5]
```
