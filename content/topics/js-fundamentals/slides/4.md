### Conditional Logic

- equality operators
- relational operators
- logical operators

---

#### Equality Operators

- `==`: equality
- `!=`: inequality
- `===`: identity
- `!==`: nonidentity

___

equality & inequality examples:

```js
5 == 5 // true
5 == "5" // true
5 != 5 // false
5 != "5" // false
```

identity & nonidentity examples:

```js
5 === 5 // true
5 === "5" // false
5 !== 5 // false
5 !== "5" // false
```

---

#### Relational Operators

- `>`: greater than
- `<`: less than
- `>=`: greater than or equal to
- `<=`: less than or equal to

___

greater than & less than examples:

```js
5 > 5 // false
5 < 5 // false
5 > 4 // true
4 < 5 // true
```

greater than or equal to & less than or equal to examples:

```js
5 >= 5 // true
5 <= 5 // true
5 >= 4 // true
4 <= 5 // true
```

---

#### Logical Operators

- `&&`: and
- `||`: or

___

logical and examples:

```js
(5 > 8) && (5 > 6) // (false) && (false) -> false
(5 > 4) && (5 > 5) // (true) && (false) -> false
(5 > 9) && (5 > 4) // (false) && (true) -> false
(5 > 3) && (5 > 4) // (true) && (true) -> true
```

logical or examples:

```js
(5 > 8) || (5 > 6) // (false) || (false) -> false
(5 > 4) || (5 > 5) // (true) || (false) -> true
(5 > 9) || (5 > 4) // (false) || (true) -> true
(5 > 3) || (5 > 4) // (true) || (true) -> true
```

### Conditional Statements

- `if`
- `else`
- `else if`

---

#### `if`

```js
if (x > 5) { // if the value of x is greater than 5
  // do something
}
```

The code inside the if block is only executed if the condition is satisfied.

---

#### `else`

```js
if (x > 5) { // if the value of x is greater than 5
  // do something
}
else { // if the original if statement was not true
  // do something
}
```

If the original `if` statement is not satisfied the code inside the else block will be executed.

One of the branches **must** execute.

---

#### `else if`

```js
if (x > 5) { // if the value of x is greater than 5

}
else if (x < 2) { // if the value of x is less than 2 and the preceding if statement was false

}
```

If the original `if` statement is not satisfied the conditional statement in the following `else if` statement resolves.

If the `else if` condition is satisfied the `else if` block of code executes.

___

#### Chaining `else if` statements

You can add any number of additional `else if` statements to an existing `if` statement.

```js
if (a) { // if a resolves to true

}
else if (b) { // if b resolves to true and the preceding if statement was false

}
else if (c) { // if c resolves to true and the preceding if statement was false

}
```

`else if` chains always resolve in order, and only one code block can execute.

___

#### Extending `else if` with `else`

An `else` statement can be added after any number of `else if` statements.

```js
if (a) { // if a resolves to true

}
else if (b) { // if b resolves to true and the preceding if statement was false

}
else if (c) { // if c resolves to true and the preceding if statement was false

}
else { // if ALL preceding if statements were false

}
```
