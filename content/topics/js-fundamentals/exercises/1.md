You were introduced to the five programming fundamentals in JavaScript and learned about some of the idiosyncrasies of JavaScript. Your goal for the remainder of this topic is to gain more familiarity with the JavaScript language fundamentals. The easiest way to do this is by solving small problems that require you to practice the fundamentals in relative isolation.

### Collection One

You can start practicing by cloning this [live coding repository](https://github.com/pdmxdd/live-coding) that includes some simple problems that can be solved with pure programming fundamentals.

### Collection Two

After completing the previously linked problems you can find more problems on Codewars.com. [Follow this link](https://www.codewars.com/kata/search/javascript?q=&r[]=-8&r[]=-7&r[]=-6&beta=false) for a pre-formatted search of relatively easy JavaScript problems.

{{% notice note %}}
Using Codewars will require you to create a free account with their platform. We prefer Codewars because their platform provides a live coding environment in the browser, allowing you to write your code and test your solution all from the browser. It's a rather intuitive platform, but ask your instructor for help if it gives you problems.
{{% /notice %}}
